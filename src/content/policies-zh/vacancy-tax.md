## 问题

丹佛有数千套住房空置——被投机者持有，等待价格上涨；被投资者视为金融资产；或仅仅是被疏忽的业主遗弃。与此同时，7500人无家可归，还有数千人面临一次租金上涨就可能流离失所的困境。

在有人居住的社区中，空置房屋会吸引犯罪，降低邻居的房产价值，并且在一个面临住房危机的城市中，这在道德上是令人无法容忍的。

### 丹佛目前的做法

丹佛目前**对空置住宅物业没有任何空置税或处罚**。美国人口普查局的美国社区调查（2022年）估计，丹佛的住宅空置率约为5-7%，这意味着有5000-8000套空置房屋。其中一些处于正常的“待售”或“待租”流程中，但相当一部分被空置用于投机或仅仅是被遗弃。

**重要的法律说明：TABOR（科罗拉多州纳税人权利法案）。** 科罗拉多州纳税人权利法案（科罗拉多州宪法第X条第20款）要求任何新税种都必须经过选民批准。丹佛的空置税需要作为一项投票措施提交给选民。这并非障碍，而是一个民主程序。丹佛选民已经多次批准了支持进步事业的投票措施（2018年的市政宽带，2022年的第123号提案，用于住房）。将收入用于经济适用房的空置税是一项强有力的投票提案。

## 我们的解决方案

### 累进空置税

对日历年中空置超过6个月的住宅物业征税，并随着时间的推移而增加：

| 空置年限 | 税率（评估价值的百分比） |
| -------- | ---------------------- |
| 第1年     | 1%                     |
| 第2年     | 2%                     |
| 第3年     | 3%                     |
| 第4年     | 4%                     |
| 第5年及以上 | 5%                     |

**对于评估价值为$500,000的房屋：**

- 空置第1年：$5,000税
- 空置第3年：$15,000税
- 空置第5年及以上：$25,000/年

这使得投机变得越来越昂贵，同时给予业主合理的装修或出售时间。

**豁免（范围窄且具体）：**

- 正在进行有效许可证的装修（最长12个月）
- 业主正在接受医疗护理或军事部署（有证明文件）
- 房产以公平市场价值积极挂牌出售（最长6个月）
- 遗产正在遗嘱认证中（最长12个月）

**以下情况不予豁免：**

- 没有许可证的“计划装修”
- 没有时间表的持有以待未来开发
- 公司投资组合管理
- 季节性/度假屋（如果业主没有丹佛主要住所）

**执行：** 需要每年进行空置登记。城市检查员通过公用事业使用数据（水、电）、实地检查和邻居报告来验证空置情况。未登记：自动按5%的税率征收。

**收入：** 所有空置税收入都将用于丹佛经济适用房信托基金，专门用于社会住房收购和社区土地信托。

**预计收入：** $5000万-$8000万/年（基于估计的5000-8000套空置房屋和不断上涨的税率）

### 国际证据：空置税有效

| 城市/国家                 | 模式                                                              | 结果                                                                                                                                            |
| ------------------------- | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| **加拿大温哥华**          | 1%空屋税（2017年），增加到3%（2021年）。                          | 第一年空置率下降25%。累计收入超过$1.7亿加元（2017-2023年）。数千套房屋重返租赁市场。                                                                         |
| **澳大利亚墨尔本**          | 对空置6个月以上的房产征收1%的空置住宅土地税。                       | 收入用于经济适用房。空置房产明显减少。                                                                                                                  |
| **法国巴黎**              | _Taxe sur les logements vacants_ - 对空置1年以上的房屋征收累进税。 | 巴黎的空置房屋减少。收入用于社会住房建设。                                                                                                                |
| **英国（多个议会）**       | 对长期空置房产征收高达300%的市政税附加费。                        | 数千套空置房屋被重新投入使用。                                                                                                                          |
| **美国华盛顿特区 (Washington, DC)** | 空置房产税是标准税率的5倍。                                     | 是美国最高的空置税率之一。抑制投机和囤地。                                                                                                                    |

**模式很清晰：** 对空置房产征税的城市始终看到空置率下降和住房供应增加。这是最简单、最有效的住房政策工具之一。

## 资金来源

这是一项创收措施，而不是一项成本。行政管理成本（$100万-$200万/年）可以通过产生的收入多次覆盖。

## 常见问题

**“那么财产权呢？”**
您有权拥有财产。您无权在危机期间囤积住房。城市对房产征税以用于学校、道路和服务。这是相同的原则：您的空置房产给社区带来成本，社区有权获得补偿。

**“业主难道不会声称该房产已被占用吗？”**
公用事业数据不会说谎。一个6个月内用水量为零且用电量极低的单元是空置的。结合实地检查和邻居报告，逃避行为很难实现，并且虚假报告的处罚非常严厉。

**“这会伤害正在装修的人。”**
凭有效许可证的12个月装修豁免涵盖了合法的装修。如果您的“装修”耗时3年且没有许可证，那不是装修，而是投机。
